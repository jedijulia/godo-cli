# godo-cli

**note: work in progress**

just another todo cli app. go do things!

```
[adding to a todo]
todo add "item" #group #group2

[list all todos]
todo

[displaying a group of todos]
todo #group

[mark todos as done]
todo done 0
todo done #group
todo undone 0
todo undone #group

[delete todos]
todo delete 0
todo delete #group
todo clear

[hide done todos]
todo clean
todo clean #group
```
